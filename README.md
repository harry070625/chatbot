# 🤖 AI Chatbot - King Arthur Baking Mixes

An advanced AI chatbot system for King Arthur Baking products using LangGraph and Ollama.

## ✨ Features
- Web scraping of King Arthur Baking mixes
- Vector embeddings with OpenAI
- LangGraph agent with advanced reasoning
- Beautiful Streamlit UI
- Ollama local LLM support
- PostgreSQL + ChromaDB storage
- Agent graph visualization

## 🚀 Quick Start
```bash
docker-compose up -d
```
